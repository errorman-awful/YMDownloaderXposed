# Xposed модуль для загрузки музыки Yandex Music
## Описание
Модуль перехватывает текущий выбранный трек и выводит overlay кнопку поверх интерфейса для скачивания.
После скачивания файл можно найти в папке Music/YandexMusic или поделиться путем нажатия на ту же кнопку



## Поддерживаемые версии
Все последние и будущие версии Яндекс Музыки

## Установка
Устанавливаем модуль,включаем его для яндекс музыки,перезагружаемся,профит
